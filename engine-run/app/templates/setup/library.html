Music library path

The path your Plex server sees for the Plexify music folder — the container path you mapped the plexify-music folder to in your Plex mounts. With -v /volume3/MediaVolume3/plexify-music:/media/vol3/Music, this is /media/vol3/Music. Plexify uses it to map acquired FLACs to where Plex indexes them.
