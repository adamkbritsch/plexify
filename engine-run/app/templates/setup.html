One-time Spotify setup

Spotify requires a developer app to use its API. This is free and takes 30 seconds.

  1. Open developer.spotify.com/dashboard and log in.
  2. Click Create app. Name it anything (e.g. "Plexify"). Description optional.
  3. For Redirect URI, paste exactly:
    {{ redirect_uri }} If you change PUBLIC_BASE_URL in .env, update this here too.
  4. API: Web API. Save.
  5. On the app page → Settings, copy the Client ID and Client Secret below.
